# خارطة الطريق — Quran Pipeline

## المرحلة 1: إتقان الأساس (مكتمل ✅)
- [x] تلاوة قرآنية يومية (Shorts ×2 + فيديو طويل أسبوعي)
- [x] اختيار عشوائي موزون للقراء (عبدالباسط، العفاسي، ماهر، الغامدي)
- [x] AI فقط للعنوان والوصف (لا يمس النص القرآني)
- [x] خلفيات فيديو متحركة مختلفة لكل آية
- [x] جدولة نشر ذكية (تحليل أوقات الذروة)
- [x] نشر على يوتيوب مع احترام Quota

## المرحلة 2: مصادر إسلامية موثوقة (قيد التفكير)

### أسماء الله الحسنى — سلسلة 99
- مصدر: أسماء الله الثابتة في القرآن والسنة
- شرح كل اسم من كتب ابن عثيمين / السعدي (نصوص مدخلة يدوياً، لا AI)
- كل فيديو: اسم + معناه + دليله (آية أو حديث)

### قصص القرآن — كما وردت
- النص من القرآن فقط، بدون تفسير أو إضافة
- قصة نبي أو أمة في 30-60 ثانية (نفس pipeline التلاوة)
- التعليق: فقط رقم الآية والسورة

### أدعية القرآن
- فقط الأدعية المذكورة نصاً في القرآن (ربنا آتنا...، ربنا لا تزغ قلوبنا...)
- كل فيديو: دعاء واحد + تلاوته + الآية مكتوبة

### أحاديث الأربعين النووية
- سلسلة 42 حديثاً مشهوراً
- مصدر: متن الأربعين النووية (نصوص مدخلة يدوياً)
- كل حديث في فيديو قصير مع تلاوة اختيارية

## مبادئ صارمة
1. **لا AI في النص الديني** — القرآن، الحديث، الدعاء: كلها نصوص ثابتة
2. **لا تفسير بالرأي** — أي شرح لازم يكون من عالم معتمد أو كتاب معروف
3. **لا موسيقى** — المحتوى الإسلامي بدون موسيقى خلفية
4. **مصدر مفتوح** — كل النصوص المستعملة موثقة المصدر

## تقنياً
- كل سلسلة جديدة = Queue منفصلة في BullMQ
- قوالب بصرية مختلفة حسب نوع المحتوى
- إمكانية قنوات يوتيوب متعددة (قناة لكل سلسلة)
