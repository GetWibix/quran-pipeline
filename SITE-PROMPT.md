# 🕌 برومنت إنشاء موقع NOR Quran — منصة نشر القرآن الكريم

> هذا الملف هو المرجع الكامل لبناء الموقع التعريفي لمشروع **NOR Quran** (quran-pipeline).  
> يحتوي على كل التفاصيل: الرؤية، الصفحات، التصميم، الميزات، التقنيات، والمحتوى.

---

## 1. الرؤية العامة (Vision)

**موقع تعريفي متكامل لمشروع NOR Quran** — نظام آلي ذكي لنشر آيات القرآن الكريم على يوتيوب والمنصات الأخرى.

الموقع يجب أن يكون:
- **احترافي وجذاب** — يعكس جمال القرآن وقدسية المشروع
- **غامق (Dark Mode)** — يتماشى مع الهوية البصرية الحالية (تدرجات زرقاء/بنفسجية)
- **ناطق بالعربية الفصحى** — الجمهور المستهدف عربي
- **سريع ومجاني** — يستضاف على GitHub Pages مع Astro/Next.js
- **يشرح كل شيء** — من الفكرة إلى التنفيذ، الميزات، التقنيات، طريقة التشغيل

- **الرؤية** — نشر القرآن في كل مكان، بدون جهد يدوي
- **القيم** — مجاني، خيري، مفتوح المصدر، احترام القرآن

### 3.5 القرآن مباشر (`/quran`)

عرض القرآن الكريم من API مجاني:
- سور/list
- آيات السورة مع التفسير
- البحث
- تشغيل الصوت (من EveryAyah.com)
- وضع القراءة الليلي

**API مقترحة:**
- `https://api.alquran.cloud/v1/surah` — قائمة السور
- `https://api.alquran.cloud/v1/surah/{number}` — آيات السورة
- `https://api.alquran.cloud/v1/surah/{number}/en.asad` — ترجمة إنجليزية
- `https://audio.everyayah.com/` — ملفات الصوت

- هل يدعم ترجمة القرآن؟ حالياً بالعربية فقط
- كيف أضيف قارئ جديد؟ يعدل في الكود

## 6. محتوى كل صفحة (نصوص جاهزة)

### الرئيسية — Hero Section
```
العنوان: NOR Quran
الجملة: ننشر القرآن الكريم آلياً — 24 ساعة، 7 أيام، 365 يوم
الوصف: نظام ذكي مفتوح المصدر يختار الآيات، يولد فيديوهات، وينشرها على 
يوتيوب وفيسبوك وإنستغرام وثريدز — بدون تدخل يدوي.
زر: ابدا الآن ← /docs
زر ثانوي: شاهد الميزات ← /features
```

### الرئيسية — نبذة سريعة
```
NOR Quran يقرأ القرآن كاملاً (6,236 آية) بتسلسل بدون تكرار. 
يستخدم الذكاء الاصطناعي لاختيار الآيات المناسبة وتوليد عناوين محترمة.
ينشر في أفضل الأوقات بناءً على تحليلات الأداء.
```

### عن المشروع — القصة
```
فكرة NOR Quran انطلقت من رغبة في نشر القرآن على أوسع نطاق 
باستخدام التقنية. بدلاً من النشر اليدوي المتعب، 
قمنا ببناء نظام يعمل 24/7 يختار أجمل الآيات، 
يصمم فيديوهات جذابة، وينشرها في التوقيت المثالي.
كل هذا مجاناً ولوجه الله.
```

### الميزات — النشر التلقائي
```
📹 النشر التلقائي ليوتيوب
يرفع الفيديو إلى YouTube عبر API v3 مع جدولة النشر.
يدعم الفيديوهات القصيرة (Shorts) والطويلة.
يضبط الخصوصية تلقائياً (خاص → عام حسب الوقت المحدد).
يختار الصورة المصغرة (Thumbnail) إن وُجدت.
```

### الميزات — الذكاء الاصطناعي
```
🤖 عناوين ووصف بالذكاء الاصطناعي
يستخدم OpenRouter (نماذج مجانية) لتوليد:
- عنوان جذاب ومحترم (أقصى 70 حرف)
- وصف شامل مع تفاصيل السورة والآيات
- هاشتاغات هرمية (عام → سورة → قارئ)
- 5 أنماط مختلفة للعناوين مع تتبع الأداء
كل هذا بدون مبالغة أو خداع — أدب مع كلام الله.
```

### الميزات — أوقات النشر المثلى
```
📊 تحليلات وأوقات النشر المثلى
يختار وقت النشر بذكاء:
- 70%: ينشر في أفضل وقت معروف (أعلى تفاعل)
- 30%: يجرب أوقات جديدة لتحسين الأداء
بعد 48 ساعة من النشر، يحلل النتائج ويحدث قاعدة البيانات.
مع الوقت، النظام يتعلم أفضل أوقات النشر لكل نوع محتوى.
```

### الميزات — منصات متعددة
```
🌐 نشر متعدد المنصات
بعد رفع الفيديو لـ YouTube، ينشر تلقائياً على:
- Facebook (صفحة)
- Instagram (حساب تجاري)
- Threads
مع إشعار عبر Telegram عند النجاح أو الفشل.
```

## 8. اعتبارات الأداء و SEO

- **Lighthouse 90+** في كل المقاييس
- **Open Graph tags** لكل صفحة (Facebook, Twitter, Telegram)
- **JSON-LD** لهيكلة البيانات (SoftwareApplication, Quran)
- **Sitemap.xml** للموقع كامل
- **Robots.txt** للسماح بالزحف

- **PWA (Progressive Web App)** — اختياري، للتحميل كتطبيق
- **تحميل الصور بشكل كسول (lazy loading)**
- **تقليل JavaScript** — قدر الإمكان static

## 13. ملاحظات مهمة للمطور

1. **الموقع تعريفي فقط** — لا يحتوي على بيانات حساسة
2. **آيات القرآن** — تجلب من API مباشرة، لا تخزن محلياً
3. **لا يحتاج باك إند** — كله static + API calls من المتصفح
4. **CNAME للموقع** — `quran.waxbix.com`
5. **النشر** — على GitHub Pages (gh-pages branch أو ريبو منفصل)
6. **التصميم** — احترام كلام الله، لا إعلانات، لا clickbait
7. **المحتوى** — كل النصوص بالعربية الفصحى
8. **التواصل** — `contact@waxbix.com` للاستفسارات
