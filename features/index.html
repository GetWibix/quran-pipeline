الميزات

كل ما تحتاج لنشر القرآن الكريم آلياً باحترافية

النشر التلقائي ليوتيوب

يرفع الفيديو إلى YouTube عبر API v3 مع جدولة النشر. يدعم الفيديوهات القصيرة (Shorts) والطويلة. يضبط الخصوصية تلقائياً (خاص ← عام حسب الوقت المحدد). يختار الصورة المصغرة (Thumbnail) إن وُجدت.

عناوين ووصف بالذكاء الاصطناعي

يستخدم OpenRouter (نماذج مجانية) لتوليد: عنوان جذاب ومحترم (أقصى 70 حرف)، وصف شامل مع تفاصيل السورة والآيات، هاشتاغات هرمية (عام ← سورة ← قارئ)، و5 أنماط مختلفة للعناوين مع تتبع الأداء. كل هذا بدون مبالغة أو خداع — أدب مع كلام الله.

تحليلات وأوقات النشر المثلى

يختار وقت النشر بذكاء: 70% ينشر في أفضل وقت معروف (أعلى تفاعل)، 30% يجرب أوقات جديدة لتحسين الأداء. بعد 48 ساعة من النشر، يحلل النتائج ويحدث قاعدة البيانات. مع الوقت، النظام يتعلم أفضل أوقات النشر لكل نوع محتوى.

نشر متعدد المنصات

بعد رفع الفيديو لـ YouTube، ينشر تلقائياً على Facebook (صفحة)، Instagram (حساب تجاري)، وThreads. مع إشعار عبر Telegram عند النجاح أو الفشل.

قراءة متسلسلة بدون تكرار

يقرأ القرآن كاملاً (6,236 آية) بتسلسل من الفاتحة للناس بدون تكرار. يتتبع الآيات التي تم نشرها بالفعل لضمان تغطية كاملة للقرآن.

أصوات عدة قراء

نظام توزيع عشوائي: عبد الباسط 40%، العفاسي 30%، المعيقلي 20%، الغامدي 10%. يختار قارئاً مختلفاً لكل فيديو لتنويع المحتوى.

التقارير والإحصائيات

إحصائيات يومية عبر Telegram بعد كل نشر: عدد المشاهدات، الإعجابات، التعليقات، المشاركات، وأداء الهاشتاغات.

حد الأمان (API Safety)

9,500/10,000 وحدة API — حماية من الاستنزاف. يوقف النشر تلقائياً عند الاقتراب من الحد الأقصى لتفادي التكاليف غير المتوقعة.
